Build reports from scratch:

```javascript
// Example custom report configuration
{
  "report_name": "Q4 2024 AI Performance Review",
  "sections": [
    {
      "type": "executive_summary",
      "metrics": ["overall_score", "visibility", "sentiment"]
    },
    {
      "type": "trend_analysis",
      "period": "90d",
      "comparison": "previous_period"
    },
    {
      "type": "competitive_analysis",
      "competitors": ["Brand A", "Brand B", "Brand C"],
      "metrics": ["visibility", "authority"]
    }
  ],
  "format": "pdf",
  "branding": true
}
```

## Sharing & Collaboration

### Share Settings

Control how reports are shared:

<Warning>
  Shared reports respect user permissions - viewers only see data they have access to.
</Warning>

- **Public Link**: Anyone with link can view
- **Password Protected**: Requires password for access
- **Team Only**: Limited to team members
- **Time-Limited**: Links expire after set duration

- **Export Permissions**: Control download access

## API Access

Programmatically generate and access reports:

```bash
# Generate a report via API
curl -X POST "https://api.friction.com/v1/reports" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "template": "executive_summary",
    "date_range": "last_30_days",
    "format": "json"
  }'

# Response
{
  "report_id": "rpt_abc123",
  "status": "processing",
  "estimated_time": 30,
  "webhook_url": "https://api.friction.com/v1/reports/rpt_abc123"
}
```

## Best Practices

### Report Design

1. **Start with objectives**: What decisions will this report inform?
2. **Focus on actionable insights**: Highlight what can be improved
3. **Use appropriate visualizations**: Match chart type to data type
4. **Provide context**: Include benchmarks and comparisons
5. **Keep it concise**: Lead with key findings

### Distribution Strategy

- **Executives**: Weekly summaries, monthly deep dives
- **Marketing**: Daily alerts, weekly performance reports
- **Product**: Feature-specific reports, user sentiment analysis
- **Sales**: Competitive intelligence, market positioning
